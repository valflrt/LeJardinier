# LeJardinier

A simple and fast discord bot (french language only).

# Installation

Install the dependencies using:
```npm install```

## config

Here is an example of config.json:
```json
{
	"token": "your token goes here",
	"username": "Le Jardinier",
	"prefix": "!",
	"activity": {
		"activity": {
			"name": "!help"
		},
		"status": "online"
	}
}
```
